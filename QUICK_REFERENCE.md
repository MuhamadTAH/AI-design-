## 💡 Pro Tips

1. **Always test locally first**
   ```bash
   npm run test:local
   ```

2. **Use design examples**
   ```bash
   npm run example:design-brief
   ```

3. **Start simple, iterate complex**
   - First: Get basic mockup
   - Then: Add interactions
   - Finally: Polish details

4. **Keep API key safe**
   - Never commit `.env`
   - Use environment variables
   - Regenerate if compromised

5. **Export multiple formats**
   - HTML for web
   - MP4 for video
   - PDF for print
   - PPTX for presentations
